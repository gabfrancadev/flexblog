# Flexblog

Flexblog is a project developed during the Origamid front-end course with the goal of practicing HTML and CSS Flexbox.

The main focus of this project is understanding how Flexbox works to build flexible layouts, align elements, and distribute space between components in a clean and organized way.

## Technologies Used
- HTML5
- CSS3
- CSS Flexbox

## Project Features
- Multi-section layout
- Flexbox-based structure
- Alignment and spacing using Flexbox properties
- Clean and organized HTML structure
- No JavaScript used

## What I Learned
- How Flexbox works and its main properties
- Building flexible layouts with display: flex
- Aligning and distributing elements efficiently
- Structuring sections using semantic HTML
- Improving CSS organization for layout projects
